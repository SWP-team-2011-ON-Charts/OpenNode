User guide

 

Hello User!

OpenNode Charts is a prototype for managing virtual machines(VM). Here is a little guide for helping the use of the prototype. We hope that You will find the solutions and anwers to Your questions and problems:

·         Logging on

When entering the web-site, a login screen is displayed. For logging on You need to enter the name of the management server, username, password and then click "register". When the logging is successful, the graph and users list are displayed.

·         Layout

There are two options for displaying the graph layout - whether as a circle around the datacenter or different levels of machines under datacenter. The layout can be chosen by clicking on the "Switch layout" button.

·         Adding and removing users

You can add users with the button "Add user" on the top of the right-side panel. It is possible to specify the user name and the rights to computes. The created user will appear in the right-side table. It is possible to delete the user by clicking the red minus next to the user. The user rights can be changed when clicking the user icon - the rights to computes and whether to show or not to show the rights can be set there.

·         User rights

When user has rights to the machine or virtual machine, then the way to the allowed machines is painted with the same color that is set to the user in the right-side menu.

·         Datacenters

A new datacenter can be added by clicking the button "Register datacenter" in the left upper side. It is possible to hide/show the subordinate machines by the little button(-/+) under the datacenters icon. It is not possible to delete a datacenter at the moment.

·         Machines

When the graph is zoomed in, the machine data is displayed under the machine as a diagram. When holding the cursor on the diagram some additional information about the machine is displayed. When clicking on the diagram the information box is opened - then it is possible to create virtual machines under the selected machine and also stop/run and suspend/resume the machine. Creating a new virtual machine can be done with the button "Create VM". It is also possible to hide/show the machines(and the connected virtual machines) as the datacenters with the little button(-/+) under the icon. The machines can´t be deleted.

·         Virtual machines

The virtual machines can be created when using the button "Create VM" in the machine information box. The additional information about the virtual machine can be specified in the VM registration box. When the graph is zoomed in, the virtual machine data is displayed under the virtual machine as a diagram. When holding the cursor on the diagram some additional information about the virtual machine is displayed. When clicking on the diagram the information box is opened - then it is possible to stop/run and suspend/resume the machine. Also the virtual machines can be deleted with the button "delete VM" that is in the virtual machine information box. It is possible to move the virtual machines from one machine under another - that can be done by clicking on the virtual machines icon and then holding the mouse button down and dragging the virtual machine icon on the wanted machine icon.
