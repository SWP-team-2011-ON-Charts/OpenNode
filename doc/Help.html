Hello User!

OpenNode Charts is a prototype for managing virtual machines(VM). Here is a little guide to help You use the prototype. We hope that You will find the solutions and anwers to Your problems and questions:

  • Logging on

Entering the web-site, a login screen is displayed. To log on You need to enter the name of the management server, username, password and then click "register". If the logon is successful, the graph and users list are displayed. If the logon fails, appropriate message is displayed.

  • Layout

There are two possible graph layouts - as a circle of machines around the datacenter or as a pyramid of machines with datacenter on top. Layout can be chosen by clicking on the "Switch layout" button.

  • Adding and removing users

You can add users with the button "Add user" on the top of the right-side panel. It is possible to specify the user name and the rights to computes. The created user will appear in the right-side table. It is possible to delete the user by clicking the red minus sign next to the user. User's rights can be changed when clicking on the user icon - the rights to computes and whether to show or not to show the rights can be set there.

  • User rights

When the selected user has rights to a physical machine or a virtual machine, the way to the specified machines is colored red.

  • Datacenters

A new datacenter can be added by clicking the button "Register datacenter" located in the top-left corner. It is possible to hide/show the subordinate machines by clicking the little button(-/+) under the datacenter's icon. It is not possible to delete a datacenter at the moment.

  • Physical Machines

When the graph is zoomed in, data about the physical machine is displayed under the machine as a diagram. When holding the cursor on the diagram some additional information about the physical machine is displayed. When clicking on the diagram the information box is opened - there it is possible to create virtual machines under the selected physical machine and also stop/run and suspend/resume the physical machine. Creating a new virtual machine can be done with the button "Create VM". It is also possible to hide/show the physical machines(and the connected virtual machines) clicking the little button(-/+) under the icon. Physical machines cannot be deleted.

  • Virtual machines

Virtual machines can be created using the button "Create VM" in the physical machine's information box. Additional information about the virtual machine can be specified in the VM registration box. When the graph is zoomed in, the virtual machine's data is displayed under the virtual machine as a diagram. When holding the cursor on the diagram, some additional information about the virtual machine is displayed. When clicking on the diagram the information box is opened - there it is possible to stop/run and suspend/resume the machine. Also virtual machines can be deleted with the button "delete VM" that is in the virtual machine's information box. It is possible to move the virtual machines from one physical machine to another by clicking on the virtual machine's icon and dragging it onto the wanted machine icon holding the mouse button down.
